# ispmini

Минимальны набор контейнеров для запуска платформы с админкой

Один раз выполнить
```
docker login nexus.txix.ru
```

Поднять кластер
```
docker compose up -d
```

Админка и прокси для АПИ доступны тут
```
http://localhost:8001
```
